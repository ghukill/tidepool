# tidepool

## Installation

Clone repository:
```shell
git clone https://github.com/ghukill/tidepool.git
cd tidepool
```
  * _NOTE_: if cloning as a functional repository, consider renaming on clone

Create python environment:
```shell
uv venv --python 3.13 .venv
```

Install dependencies:
```shell
uv sync
```
